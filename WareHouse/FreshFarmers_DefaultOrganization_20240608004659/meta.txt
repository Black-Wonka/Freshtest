Task:
Develop a web and mobile app for ordering fresh agricultural products from local producers within a 50km radius. Key Requirements: 1. Frontend: User-friendly, responsive interface. Browse products, place orders, and manage orders. Search/filter products by categories, price, dietary preferences. Interactive map with toggle list view of farms within 50km. 2. Backend: Manage order lifecycle: creation, modification, cancellation, status updates. Secure user authentication and account management. Real-time inventory management integration. Payment processing with multiple options. Geolocation services for personalized recommendations. Scheduling system for pickup dates/times. Notification system for order updates via email/SMS. Performance optimization and scalability. 3. Features: Profile pages for farmers with bio, products, images. Cart management with multi-farm order alerts. User profiles: manage bio, orders, wallet, product availability, and pictures. 4. Gamification: Develop a tool to gamify the app by simulating customer savings compared to traditional shopping, CO2 reduction, and health benefits. 5. GitHub Integration: Use the GitHub repository at https://github.com/Black-Wonka/Freshtest for version control and collaboration. Project Name: FreshFarmers Overview: Connect consumers with local farmers for fresh, locally-sourced agricultural products. Pages: Landing: Login as Farmer or Customer. Home: Map and list view of nearby farms. Profile: Detailed farm profiles and shop interface. Cart: Alert for multi-farm orders, single reference number. User Management: Farmers: Manage bio, orders, wallet, product availability, profile pictures. Customers: View order history, track orders, manage delivery preferences.
